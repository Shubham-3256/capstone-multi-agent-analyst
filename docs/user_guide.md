# User Guide

This guide walks you through using the Multi-Agent AI Data Analyst web dashboard.

---

## 1. Dashboard Walkthrough

The interface is organized into a multi-page sidebar layout:
1.  **🏠 Home**: System overview and current runtime configurations.
2.  **📥 Data Ingestion**: File uploader page accepting CSV, XLSX, and JSON files. Parses structure, row counts, and detects schema definitions.
3.  **🧠 Data Intelligence**: Computes null statistics, duplicates, feature cardinality, and displays an interactive data profile table.
4.  **🛠️ Feature Engineering**: Auto-configures encoding, numerical scaling, and target-correlated feature selections.
5.  **🏋️ Model Training**: Automatically splits dataset, tunes candidate algorithms, evaluates scores, and displays comparative metrics.
6.  **📊 Visualizations**: Interactive correlation heatmaps, missing value charts, and distribution plots.
7.  **💡 Business Insights & Reports**: Provides GenAI-generated executive findings and compiles downloadable reports in PDF, DOCX, and HTML.

---

## 2. Step-by-Step Workflow Guide

### Step 1: Upload a Dataset
1.  Navigate to the **Data Ingestion** page.
2.  Drag and drop your dataset (e.g., `titanic.csv` or `housing.xlsx`).
3.  The system calculates file hash details, checks integrity, and registers it in the local database.

### Step 2: Configure Target and Run Profiler
1.  Go to the **Data Intelligence** page.
2.  Select the **Target Column** you wish to predict (e.g., `Survived` for Titanic, or `MedHouseValue` for housing).
3.  Click **Execute Data Intelligence Profile**. The profiler detects whether it is a Regression or Classification task.

### Step 3: Run Feature Engineering & ML Training
1.  Go to **Feature Engineering** to see preprocessing actions.
2.  Move to **Model Training** and select your algorithms (or select "All" to execute a competitive sweep).
3.  Click **Train Models**. Review the metric charts (Accuracy, Precision, F1, RMSE, R²).

### Step 4: Explore Charts & Export Reports
1.  Go to **Visualizations** to explore the data correlations.
2.  Go to **Business Insights & Reports** to review narrative summaries generated by the LLM.
3.  Click **Generate Executive Report** to compile and download standard PDF, Word, or HTML documents containing all findings and charts.
